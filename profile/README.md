# Apact, Inc.

**Enterprise Energy Commodities Trading Platform**

Apact is building the modern infrastructure for trading renewable energy credits, biofuels, and carbon offsets. Our platform streamlines the complex workflows of energy commodity trading with enterprise-grade security and compliance.

---

## Platform Overview

| Product | Description | Status |
|---------|-------------|--------|
| **Trading Platform** | Full-featured trade execution, counterparty management, and settlement | Production |
| **API Documentation** | Interactive API docs with AI-powered search | Production |
| **Marketing Site** | Public-facing website | Production |
| **Back Office** | Internal operations tooling | Development |

## Technology

- **Frontend:** Next.js 15, React 19, TypeScript 5, Tailwind CSS 4
- **Backend:** Supabase (PostgreSQL), Edge Functions
- **Infrastructure:** Vercel, GitHub Actions

## Products Traded

- Renewable Identification Numbers (RINs)
- Biofuels & Ethanol
- Carbon Credits & Offsets
- Low Carbon Fuel Standard (LCFS) Credits
